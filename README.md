# LPTM

Large Pre-trained Time-series models

## Setup

```bash
# Install uv (Tested with version 0.4.*)
curl -LsSf https://astral.sh/uv/install.sh | sh

# Sync dependencies
uv sync # Use `uv sync --reinstall` to rebuild after changing source code from rust
```
